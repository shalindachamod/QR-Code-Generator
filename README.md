# 📱 Modern QR Code Generator

A fast, lightweight, and stylish QR Code Generator that allows users to create QR codes for any text or website URL instantly. This project features a modern **Glassmorphism UI** and is fully optimized for mobile devices.

---

## ✨ Features
* 🚀 **Instant Generation:** Create QR codes in real-time with a single click.
* 💎 **Glassmorphism Design:** Beautifully styled with blur effects and gradients.
* 📱 **Mobile Responsive:** Designed to look great on smartphones, tablets, and desktops.
* 🖼️ **Auto-Scaling:** The QR code display expands smoothly using CSS animations.
* ✅ **Input Validation:** Simple alert system if the input field is empty.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the web app.
* **CSS3:** Advanced styling with animations and responsive media queries.
* **JavaScript (ES6):** Logic for API integration and DOM manipulation.
* **QR Server API:** Uses the [goqr.me](https://goqr.me/api/) API for generating codes.

## 📸 How It Works
1. Enter a website URL or any text in the input box.
2. Click the **"Generate QR Code"** button.
3. The app fetches a high-quality QR image from the API and displays it with a smooth animation.
